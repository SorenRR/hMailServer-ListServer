ListServer Features


This script is design for hMailserver v5 and higher, it may work with v4, but there will be certain restrictions.


Colouring Syntax for NotePad++

For the configuration files that come with the ListServer Script, you can now view them in full colour!  There is a user-defined language file included for use with NotePad++.


List Subscription

•Define a password in the subject of the mail

•Allow list members to post to the list

•Allow to (un)subscribe to the list via email (single address or list of addresses)

•Choose which list commands are available


Poster Control

•Control who can post to the list

•Posters and Recipients do not have to be the same; for example, you have a person with two addresses - with ListServer, you can allow both addresses to post, but make it so the person only receives the message at one address

•Specify posters at three levels:  Global, Domain, List-Only


List Responses

•Define an action in case the sender is not allowed: delete or redirect to admin

•Send a success/failure notification to the sender, whether the mail passes/doesn't pass


Attachment Control

•Allow only certain extension for attachments

•Restrict the size for all attachments and each attachment for itself


Other List Features

•Modify headers (Sender, Reply-To, Return-Path) of the email

•Change the MAIL FROM: in SMTP Session

•Define a prefix added to the subject

•Add a header and/or footer to the mail (text and html)

•Apply standard setting to all list or accounts or only to certain/excluded addresses

•Use of global, domain-level, or list-level config-files

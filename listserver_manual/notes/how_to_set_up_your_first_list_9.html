How To Set Up Your First List


In order to use the ListServer, you will need to have performed a few prerequisites:


1.Make sure you have installed the ListServer Script and configured your global settings.

2.Make sure you have copied the package conents to match the directory structure to their appropriate locations.  In other words, you should have a config directory, and inside there you should have the sample domain directory called mydomain.com.

3.Make sure you have created a distribution list in your hMailServer Administrator. Also, assure that the list is set to "Public - Anyone can send to the list" and that "Require SMTP Authentication" is NOT checked.  Don't worry about the "Anyone can send to the list" part, we will tweak that with the ListServer.  So let's say you created a distribution list, and named it bobslist@bobsdomain.com.

     

4.Make a copy of the RootDir\config\mydomain.com directory, and rename the copy to your own domain name.  In our example here, it will look like this: RootDir\config\bobsdomain.com.

5.Set the name of your list-level settings file.  Remember the name of the distribution list you created? Go into the config\bobsdomain.com directory, find the file called mylist_at_mydomain_com.hms and rename it.  Now, you have to get this right, or it won't work. (There are some directions for this.) So now you should have renamed the sample file to bobslist_at_bobsdomain_com.hms.

1.Now adjust your list-level settings.  These settings only affect this particular distribution list.  You can leave all the rest at default, but you should probably at least set these particular ones.

1.Make sure to set admin=.  This address is where special messages are sent in case of certain events.

2.In that file, find this setting: allowaddresses= and specify the addresses here.  These addresses will be who can post to your list.

3.Now, check to make sure to set allowmembers=.  If you set it to true, then all of the recipients of the list will be able to post.  If you want your list to be postable only by particular folks, set this to false.  (recipients are the same as members - you specify them in hMailServer Administrator's section of distribution lists.)

6.Set your Domain-Level settings. 

1.Open up the config\bobsdomain\domainposters.txt.  This is a simple list of addresses of folks who can post to any list in this domain (assuming you have created a list-level settings file for them).  There should be one address on each line.  If this file is missing, or is blank, the script should work fine anyway.

2.The other files in the config\bobsdomain\ directory can be left as default, or you can tweak them however you like.  They are texts that are added to certain messages, such as an error message if a user is not allowed to post, or the notice a user receives if they subscribe to a list.

Directory Structure and Files

Saturday, 2 March 2013 - 13:47:43



The main script (EventHandler.vbs) should go in RootDir, in a subfolder called Events.

See Installation above for more details.


You can put the listserver_manual directory anywhere, really.  It is not used by any script.


For the other files in this package, in the RootDir folder, you will have a folder called config.  If you don't already have this directory, then copy it to your Rootdir.  Otherwise, you can copy it contents to your RootDir\config directory.


Inside the config folder you will have a folder for each email domain for which you wish to use the listserver.  The folder should have the same name as the domain.  

mydomain.com

config.txt


Now, inside the domain folder, you should have the following files.  If you want to change their names, you must also do so in the configuration section.

domainposters.txt

failure_notification.txt

footer.txt

footer_html.txt

header.txt

header_html.txt

my_list_at_mydomain_com.txt

success_notification.txt


The domainposters.txt file lets you specify addresses who are allowed to post to all the lists in that domain.

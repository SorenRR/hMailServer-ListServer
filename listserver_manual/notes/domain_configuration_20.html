Domain Configuration




Setting up the Configuration Directory


•First thing is to make sure you have the directory structure correct.

•Make sure the domain's directory is the same as the domain.

•For example, if your domain is mydomain.com, that will be the name of your domain folder.



Setting up the Domain Poster file


The Domain Posters file is a list of folks who can post to any list in the domain. This will save you the hassle of having to add your primary posters to each list configuration!


The file must be named domainposters.txt or it won't work.


Create a simple, blank text file in your domain's config directory, usually at RootDir/config/mydomain.com/


In this file, you can place a list of email addresses - one on each line.

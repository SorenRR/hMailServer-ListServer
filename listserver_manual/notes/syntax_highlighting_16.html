How To Install the ListServer Syntax Highlighter for Notepad++

This ListServer package comes with syntax colouring highlighter for Notepad++.  This will allow your config files to be coloured in a way that makes them easier to read.  All comments will be shown in gray, settings in blue, and even html will be rendered.


To install the syntax highlighter in Notepad++ version 6+:

•In the main menu, go to View > User defined language...

•Push the "Import..." button.

•Browse to the file in this package called hMailListServer.xml.

•It will report "Import Successful"

•Now, go and open one of the configuration files in Notepad++.

•Voila!  Colour!


Nota Bene:  If you leave the setting configfileextension set to .hms, then these files will automatically recieve this colouring.  If you use .txt instead, you have to change the colour to "hMailListServer" in Npp's Language menu.  


Note that if you want colouring on the EventHandlers.vbs script, you will want to use the vbs language instead, which should work automatically.

Upgrading from the Previous Version



If you are upgrading from a previous version of the ListServer script, there a couple of things to look at:


•Make sure the new global setting configfileextension is properly set.  The old configuration used .txt, but the new default is .hms.  You can use either, but if you already have a hundred config files with the .txt extension, then you should either rename all those files to the new extension, or set configfileextension to .txt.

•The global constant emailadmin has been changed to serveradmin.
